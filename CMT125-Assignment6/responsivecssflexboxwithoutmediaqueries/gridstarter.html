CSS Grid

To create a responsive layout with CSS Grid, I replaced the "flex" value of the display property with "grid." From there I used grid-template-columns with auto-fit to fill out the page. I also used minmax to assure that each column wouldn't expand beyond it's initial width nor shrink to the point that it conflicted with the content. Also, I created a separate grid to accomodate the differing content on the page.
